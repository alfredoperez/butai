## page

### closing

- **Doc Footer** (`doc-footer`) · engine: none — The close — a "what's next" band with a call to action, plus a muted footer meta line. Ends with forward momentum, not a summary.
### code

- **Code Block** (`code-block`) · engine: none — A terminal window plus a code viewer, both hand-highlighted with span classes (no highlight.js, no CDN) — a command session and a focused source excerpt.
### content

- **Card Grid** (`card-grid`) · engine: none — A responsive 2/3-column grid of cards — each a title row with status badges, a body, and a muted meta footer. The default unit for a titled item with tags and a note.
### data

- **Comparison** (`comparison`) · engine: none — A side-by-side pair — a before/other side (warm tint) versus an after/ours side (green tint), each with a mono label. The core X vs Y device.
- **Data Table** (`data-table`) · engine: none — A full-width comparison table — hairline rows, uppercase-mono headers, and two tinted compared columns (one muted, one in a status color).
- **Metric Grid** (`metric-grid`) · engine: none — A trio of counted stat cards — a mono kicker, a big number in a status color, and a caption. The scannable summary after a grouped section.
### emphasis

- **Callout** (`callout`) · engine: none — The thesis highlight — a faint accent wash behind a 1px accent border, holding the one line that matters.
- **Quote Band** (`quote-band`) · engine: none — A full-bleed brand band holding one pull-quote — light text on a tinted accent surface. Use at most once or twice per document.
### header

- **Doc Hero** (`doc-hero`) · engine: none — Brief/report opener — a mono eyebrow, one tight display headline with one accent word, a one-line summary, and a meta row of date + status badges.
### structure

- **Decisions** (`decisions`) · engine: none — A numbered locked-decisions list, ADR-style — a chip digit, a title, and a muted rationale, one row per decision.
- **Section Header** (`section-header`) · engine: none — Section marker — an uppercase mono eyebrow (the lead one takes the accent) above an h2 with a hairline underline. Paces a document into scannable sections.
- **Timeline** (`timeline`) · engine: none — A vertical stepped sequence with counter chips down the left, plus a compact horizontal flow-pipeline variant for a left-to-right process.
